# QuizMaster-Interactive-Python-Quiz-App
The project demonstrates core Python programming concepts such as object-oriented programming, API integration, data handling, and user interaction. The quiz dynamically fetches questions and evaluates user responses in real time, providing instant feedback and score tracking.

## Features

- Multiple-choice quiz questions
- Dynamic question loading (API or local data)
- Real-time score tracking
- Instant feedback on answers
- Object-Oriented Programming structure

## Tech Stack

- Python 3
- OOP (Classes & Objects)
- API Handling (Optional: Open Trivia Database)
- Command Line Interface (CLI)

Q1: Is Python dynamically typed? (True/False): True
Correct!
Your score: 1/1

- Applied Object-Oriented Programming in Python
- Managed data flow across multiple modules
- Built a user-interactive CLI application
